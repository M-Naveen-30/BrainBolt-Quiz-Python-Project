# BrainBolt ‚Äî Spark Your Mind ‚ö°

A **Python OOP mini-project** for **PyCharm** that runs a 4-topic MCQ quiz
(Python / Data Science / Java / Full Stack), generates the right
certificate (Gold / Silver / Bronze / Appreciation) from the user's score,
and delivers it via **Email (Gmail SMTP)** and **WhatsApp (pywhatkit)**.

The visual pipeline uses **Pygame** (splash + opening music + 3-2-1
countdown), images use **Pillow**, and the certificate PDF is rendered
with **ReportLab**.

---

## Pipeline (10 Steps)

| # | Step | Tech |
|---|------|------|
| 1 | Opening **logo + mild music** | Pygame |
| 2 | **Register** (name / DOB ‚Üí age / phone / email / password*) | Terminal + SQLite |
| 3 | **Login** (recursive, case-sensitive) | Terminal + SQLite |
| 4 | **WhatsApp OTP** via pywhatkit (terminal fallback) | pywhatkit |
| 5 | **Home page** with 4 topic choices | Terminal |
| 6 | **Instructions** + "ALL THE BEST!" | Terminal |
| 7 | **3-2-1 countdown** | Pygame |
| 8 | 15 **MCQ questions** (shuffled questions + options) | Terminal |
| 9 | Result + **answer review with explanations** (only for chosen topic) | Terminal |
| 10 | Auto-generated **certificate** ‚Üí emailed (Gmail SMTP) + WhatsApp summary | Pillow + ReportLab + smtplib + pywhatkit |

\*Password is case-sensitive.

## OOP Concepts (all 6, with file references)

| Concept | Where |
|---------|-------|
| **Class & Object** | `brainbolt/user.py`, `database.py`, `auth.py`, `mailer.py`, `otp_service.py`, every `quizzes/*.py` |
| **Encapsulation** | `User.__name / __password / __dob` (private), `Database.__conn` |
| **Abstraction** | `BaseQuiz` (uses `ABC` + `@abstractmethod`), `AuthService` facade |
| **Inheritance** | `PythonQuiz`, `DataScienceQuiz`, `JavaQuiz`, `FullStackQuiz` all extend `BaseQuiz`. Certificate classes inherit `BaseCertificate`. |
| **Polymorphism** | Each quiz overrides `topic_key()`, `pretty_name()`, `motivational_quote()`. Certificate subclasses set their own `tier / accent_color / medal_glyph`. |
| **Constructor & Destructor** | `Database.__init__` opens connection, `Database.__del__` closes it. |

## Setup (PyCharm)

1. **Open the folder in PyCharm**  
   `File ‚Üí Open ‚Üí BrainBolt-Quiz`

2. **Create a virtual environment** (PyCharm offers it automatically) and
   install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure Gmail** ‚Äî edit `config.ini`:
   ```ini
   [email]
   sender_email     = your_email@gmail.com
   sender_app_pass  = xxxx xxxx xxxx xxxx     ; 16-char App Password
   sender_name      = BrainBolt Team
   ```

   **How to get a Gmail App Password (free)**
   1. Go to https://myaccount.google.com/security and turn **ON** *2-Step Verification*.
   2. Visit https://myaccount.google.com/apppasswords  
   3. Choose **App = Mail**, **Device = Other ‚Üí BrainBolt** ‚Üí Generate.
   4. Copy the 16-character password (lowercase, no spaces) into `sender_app_pass`.

4. **WhatsApp prerequisite** ‚Äî make sure you have
   `https://web.whatsapp.com/` already logged in on the same machine
   that runs the script (pywhatkit drives that browser tab).

5. **First-time: build assets** (logo, music, answer PDFs):
   ```bash
   python build_assets.py
   ```

6. **Run the project**:
   ```bash
   python main.py
   ```

## Persistence

Everything is saved in **`brainbolt.db`** (SQLite):

* `users`     ‚Üí registration details of every user
* `attempts`  ‚Üí every quiz attempt (name, topic, score, certificate, taken_at)

You can open it with any SQLite browser to inspect.

---

## Notes & Tips

* If **WhatsApp send fails** (no internet, WhatsApp Web not logged in,
  etc.) the OTP is **printed in the terminal** so the pipeline never
  stalls.
* If **Gmail is not configured** in `config.ini`, the certificate is
  **still generated locally** (PDF in `output/`) and the email step is
  skipped with a clear message.
* `pyautogui` is only used after pywhatkit to focus the WA window ‚Äî it
  is safe to ignore failures on headless machines.
